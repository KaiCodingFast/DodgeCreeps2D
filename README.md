# 躲避怪物2D版，Dodge Creeps 2D



![游戏截图或动图](screenshot.png)



## 📖 简介

这是一款2D动作躲避游戏，玩家控制角色躲避不断出现的小怪，坚持越久得分越高。



## 🎮 下载与运行



### 方式一：直接下载（推荐）
- Windows版本：

前往 **[Releases](https://github.com/KaiCodingFast/DodgeCreeps2D/releases)** 页面，下载 `DodgeCreeps2D_v1.0.0_windows.zip`，解压后运行 `.exe` 文件即可。



> ⚠️ 注意：`.exe` 和 `.pck` 必须在同一目录下，否则游戏无法运行。

- Adroid版本：

前往 **[Releases](https://github.com/KaiCodingFast/DodgeCreeps2D/releases)** 页面，下载 `DodgeCreeps2D_v1.0.0_Android.zip`，解压后运行 `.apk` 文件即可。





### 方式二：从源码构建

如果你有 Godot 引擎：

1. 克隆本仓库：`git clone https://github.com/KaiCodingFast/DodgeCreeps2D.git`

2. 用 Godot 打开 `project.godot`

3. 按 `F5` 或点击“运行”按钮



## 🕹️ 操作说明
- **空格**或**Enter**：开始游戏

- **W/A/S/D** 或 **方向键**：移动角色
